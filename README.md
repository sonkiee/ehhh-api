## Overview

Ehhh is built to be simple, addictive, and scalable.  
Users can create dilemmas, vote instantly, and view live results.  
The architecture is clean and modular to support future features like comments, streaks, ads, and real-time updates.

## Tech Stack

- **Golang**
- **Chi** (HTTP router)
- **PostgreSQL**
- **pgx + sqlc**
- **JWT** authentication

## Core Features (MVP)

- User registration & login
- Create dilemma
- List dilemmas (feed)
- Vote on dilemmas
- View vote results
